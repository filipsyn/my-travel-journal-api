API Endpoints
-------------

- **Global URL prefix:** `/api/`

## User

- **URL prefix:** `/user/`

### Get All Users

- **URL:** `/`
- **Method:** `GET`
- **Requirements to access**
    - Admin Role

### Create New User

- **URL:** `/`
- **Method:** `POST`

### Get User By ID

- **URL:** `/{id}/`
- **Method:** `GET`

### Update User Information

- **URL:** `/{id}/`
- **Method:** `PATCH`

### Delete User

- **URL:** `/{id}`
- **Method:** `DELETE`
